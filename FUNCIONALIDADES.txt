🤖 WHATSAPP BOT BUILDER - FUNCIONALIDADES COMPLETAS
================================================================

📋 VISÃO GERAL
--------------
Sistema completo para criação e gerenciamento de chatbots para WhatsApp com 
interface administrativa, sistema de usuários, pagamentos e IA integrada.

🔐 SISTEMA DE AUTENTICAÇÃO
--------------------------
✅ Registro de usuários com validação de email
✅ Login seguro com hash de senhas
✅ Sistema de referência com códigos únicos
✅ Três níveis de acesso: User, Moderador, Admin
✅ Gerenciamento de sessões
✅ Proteção contra contas banidas

👥 SISTEMA DE USUÁRIOS
----------------------
✅ Perfis completos com saldo e histórico
✅ Sistema de referência com bonificações (R$ 0,50 por referência)
✅ Carteira digital integrada
✅ Histórico de ganhos por referência
✅ Gerenciamento de saldo pelos administradores

🏠 DASHBOARD PERSONALIZADO
--------------------------
✅ Estatísticas em tempo real
✅ Gráficos de performance mensal
✅ Contador de bots ativos/inativos
✅ Resumo de ganhos e referências
✅ Navegação intuitiva entre seções

🤖 CONSTRUTOR DE BOTS
---------------------
✅ Interface drag-and-drop para criação
✅ Configuração de mensagens de boas-vindas
✅ Sistema de botões personalizáveis
✅ Integração com catálogo de produtos
✅ Configurações de negócio (nome, informações)
✅ Preview em tempo real
✅ Simulator integrado para testes

🛍️ CATÁLOGO DE PRODUTOS
------------------------
✅ Cadastro ilimitado de produtos
✅ Campos: nome, descrição, preço, categoria, imagem
✅ Importação via CSV em massa
✅ Organização por categorias
✅ Busca e filtros avançados
✅ Integração automática com bots

🧠 IA CONVERSACIONAL AVANÇADA
-----------------------------
✅ Respostas contextuais baseadas nos produtos cadastrados
✅ Busca inteligente por palavras-chave
✅ Reconhecimento de intenções do usuário
✅ Respostas em português brasileiro
✅ Sistema de fallback para perguntas não reconhecidas
✅ Análise de sentimento básica

Exemplos de IA em funcionamento:
• "Quais carros tem?" → Lista todos os carros cadastrados
• "Tem iPhone?" → Busca iPhones específicos no catálogo  
• "Quanto custa?" → Mostra preços dos produtos mencionados
• "Formas de pagamento?" → Informa opções disponíveis
• "Horário de funcionamento?" → Resposta personalizada
